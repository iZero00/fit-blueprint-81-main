# BassiniFit - Plataforma de Treinos Personalizados

## Sobre o Projeto

Plataforma completa para gestão de treinos personalizados, permitindo que alunos acompanhem seu progresso e administradores gerenciem alunos e exercícios.

## Tecnologias

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Auth, Database, Storage)
- React Query
- Recharts

## Como rodar localmente

1. Clone o repositório
2. Instale as dependências:
```sh
npm install
```
3. Inicie o servidor de desenvolvimento:
```sh
npm run dev
```

## Funcionalidades

- **Alunos:**
  - Visualização de treinos diários
  - Check-in de exercícios
  - Upload de fotos de treino
  - Galeria de fotos com download
  - Acompanhamento de progresso

- **Admin:**
  - Gestão de alunos (criar, editar, inativar)
  - Gestão de exercícios e grupos musculares
  - Criação e atribuição de treinos
  - Visualização de "Visão do Aluno"
  - Acesso à galeria dos alunos
